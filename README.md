An opinionated springboot starter template.

## Getting Started

- Getting Code

```bash
git clone https://github.com/skyclouds2001/springboot-template-sky.git
```

- Install dependencies

```bash
mvn dependency:resolve
```

- Develop

```bash
mvn spring-boot:run
```

- Build

```bash
mvn clean package
```
